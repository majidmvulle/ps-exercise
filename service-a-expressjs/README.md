# service-a-expressjs

## 📖 Project description

A service that is publicly available and exposed endpoints to fetch a string from another service.

## 🔧 Installation (local)
- `git clone` the repo.
- `yarn install` to install dependencies.
- `npx ts-node app.ts` to run the webserver.
